Olá, este é o projeto: Trilha do Conhecimento, onde eu irei 
descrever minhas fases na aprendizagem do mundo JavaScrip 
por meio de um website. 

A identidade visual representa os anos 90, inspirado no jogo:
SUPER MÁRIO. Pois a programação é assim, cada etapa uma nova 
descoberta e um desafio diferente.
